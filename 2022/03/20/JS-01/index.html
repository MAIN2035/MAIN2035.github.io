编译

JavaScript是编译型语言，不同的是JavaScript的代码编译发生在代码执行前的几微秒内

LHS与RHS

LHS查找的目的为了赋值

RHS查找的目的是为了引用值

作用域

作用域查找从词法作用域（即最内部作用域）开始逐层向上查找，直到匹配到或触顶为止

全局变量会自动成为全局对象，因此可通过 window.a（a是要访问的变量）访问被同名变量屏蔽的全局变量

欺骗词法

使用欺骗词法则JavaScript引擎无法对作用域栈进行优化

eval

    /*在函数内动态创建变量b，屏蔽全局变量b*/
        function foo(str,a){
            eval(str);//动态创建代码
            console.log(a,b);
        }
        var b = 2;
        foo("var b = 3;",1);
    /*结果：1,3*/

with

/*重复引用对象的多个属性*/
        var obj = {
            a = 1,
            b = 2
        };
        with(obj){
            a = 3;
            b = 4;
            c = 5;//对象obj中没有c，返回undefined,并创建一个全局变量c = 5;
        }

函数作用域

函数中的作用域属于局部作用域，函数具有很好的闭包性，隐藏该隐藏的。

JavaScript中多

立即执行函数表达式（IIFE）

  • 两种方式
/*ONE*/
        (function foo(){
            var a = 2;
            console.log(a);
        })();
/*TWO*/  
        (function foo(){
            var a = 2;
            console.log(a);
        }());//括号位置不同哦
  • 传入参数
       /*传入window对象，引用全局变量*/
        var a = 2;
        (function (global){//匿名IIFE
            var a = 3;
            console.log(a); 
            console.log(global.a);
        })(window)
/*结果：3,2*/

提升

变量声明、函数声明均会提升到顶部。

let声明的块中的变量声明不会提升，而是保留在原地

函数先于变量提升，函数表达式不会被提升，后面的函数声明覆盖前面的函数声明

        function a(){
            console.log(1);
        }
        a = function a(){//不会被提升
            console.log(2);
        }        
        function a(){
            console.log(3);
        }
        a();
/*结果：3*/

模块

调用一个函数只返回其属性数据，而没有返回闭包对象（函数内的函数）的不是真模块

具有函数属性的对象不是真模块

/*一个模块*/
        function CoolModule(){
            var something = "so";
            var another = [1,2,3];
            function doSomething(){
                console.log(something);
            }
            function doAnother(){
                console.log(another.join("!"));
            }
            return {
                doSomething:doSomething,
                doAnother:doAnother
            };
        }
        var foo = CoolModule();
        foo.doAnother();
        foo.doSomething();


/*一个单例模式*/
        var foo = (function CoolModule(){
            var something = "so";
            var another = [1,2,3];
            function doSomething(){
                console.log(something);
            }
            function doAnother(){
                console.log(another.join("!"));
            }
            return {
                doSomething:doSomething,
                doAnother:doAnother
            };
        })();
        foo.doAnother();
        foo.doSomething();
/*结果：
1！2！3！
so
*/
